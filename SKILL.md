# Skills

Domain and technical knowledge skills used by Oku's AI agents via [Kiro](https://kiro.dev).

## technical-knowledge

Rust implementation patterns for Oku — facade design over ogun, serde-driven data pipelines, translation between urban concepts and ogun abstractions, erosion algorithms, hierarchical orchestration, and output format conversion.

## ogun-context

Reference for ogun's public API (`generate`, `Graph`, `Node`, `Edge`, `Space`, `OgunConfig`, `Layout`), the sequential logit dynamics algorithm, key parameters (β, seed, repulsion_k), potential function mechanics, and scoring metrics.

## domain-knowledge

Urban generation domain knowledge — PCB-to-city mapping, settlement morphology patterns, building category interaction matrix, arrival ordering strategies, material-based erosion cascades, hierarchical multi-scale generation, and output format design.
